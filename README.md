# YOLO Implementation in Colab

This repository contains a Jupyter Notebook demonstrating the implementation of the YOLO (You Only Look Once) object detection algorithm in Google Colab. It is intended to help users understand and experiment with YOLO for object detection tasks using the power of cloud GPUs, without local setup hassles.

## Features

- **YOLO Algorithm**: End-to-end object detection using the YOLO model architecture.
- **Google Colab**: Leverage Google Colab for free GPU resources.
- **Jupyter Notebook**: Interactive notebook for step-by-step explanation and experimentation.
- **Demo**: Detect objects in images using a pre-trained YOLO model.

## Getting Started

### Prerequisites

- Google account (to access Google Colab)
- Basic knowledge of Python and Jupyter Notebooks

### Usage

1. **Open the Notebook in Colab**  
   Click the button below to open and run the notebook directly in Google Colab:

   [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/ikramkakar/YOLO-Implementation-in-Colab/tree/main)

2. **Runtime Setup**  
   - Select `Runtime > Change runtime type` and ensure GPU is selected.
   - Run each cell in order for setup, downloading pre-trained weights, loading images, and performing detection.

3. **Customize**  
   You can upload your own images for detection or tweak parameters in the notebook.
